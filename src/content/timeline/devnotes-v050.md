AI 排行榜完成一次完整的页面与数据链路改造，从原来的价格矩阵升级为支持多视图、筛选和排序的智能数据表。

## 排行榜页面

- 新增「编程工具」与「AI 模型」两个 Tab，统一展示编程工具和模型的对比数据
- 支持表格视图与卡片视图切换
- 表格列头支持升序、降序和取消排序
- 新增列筛选，支持文本包含匹配以及价格、上下文长度等数值条件筛选
- 增加服务端首屏渲染，客户端脚本异常时仍能看到基础排行榜内容

## 模型数据

- 新增 AI 模型排行榜数据集，补充厂商、架构、参数量、激活参数量、上下文和能力字段
- 参数量优先从 DataLearner API 获取，价格、上下文长度和模型能力由 OpenRouter Models API 补充
- 新增 `scripts/sync-models.mjs`，支持全量同步、指定模型修正、API 快照和可选 Hugging Face 补充
- 增加 Kimi K3、MiniMax 系列等模型的参数量和架构信息
- 页面标注 OpenRouter 与 DataLearner 两个数据来源

## 工程升级

- 新增排行榜独立客户端模块 `src/scripts/pricing.js`
- 将项目版本升级至 `0.5.0`
